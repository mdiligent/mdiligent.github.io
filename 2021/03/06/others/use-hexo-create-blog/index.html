node.js下载及hexo配置

解决方法：我们可以先执行npm install ejs@2.7.4 --ignore-scripts后再继续后重新npm install。

把本地博客托管到Github

在github新建一个存储库，名称一定要为username.github.io（username为你的github账户名），进入settings选择githubpages，要先选择主题才能启用githubpages。更换主题，https://hexo.io/themes/ 在上边选好主题，照着做克隆下来，根据教程修改thems里的config文件好了。
